# belajar-pybind11
### Step to Using this File
1. $ git clone --recurse-submodules https://github.com/kingofaris/belajar-pybind11.git
2. $ virtualenv venv
3. $ source venv/bin/activate
4. $ pip install -r requrements.txt
5. $ mkdir build && cd build
6. $ cmake ..
7. $ make
8. $ cd..
9. $ python3 main.py
